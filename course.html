Design and Analysis of Algorithms

This course introduces the fundamentals of algorithms and data structures, focusing on their design, analysis, and application in real-world problems.

Math Basics

We started by understanding basic concepts like time and space efficiency, recurrence relations, and solved problems such as the Tower of Brahma and n-Queens problem.

Subtopics:
  • Time and Space Complexity
  • Recurrence Relations
  • Examples: Tower of Brahma, n-Queens Problem
Math Foundations

This section covers orders of growth and their importance in evaluating the efficiency of algorithms.

Subtopics:
  • Big-O Notation
  • Asymptotic Behavior
  • Order of Growth Analysis
Structured Data Management

We explored n-ary trees, binary trees, binary search trees, AVL trees, Red-Black trees, Tries, skip lists, and other structures like Fenwick Trees and heaps.

Subtopics:
  • Binary Search Trees (BST)
  • AVL Trees
  • Red-Black Trees
  • Heaps and Fenwick Trees
  • Tries and Skip Lists
Sorting and Searching Algorithms

Algorithms like Bubble Sort, Quick Sort, Heap Sort, Linear Search, Binary Search, and String Matching (KMP, Rabin-Karp) help in efficient data processing.

Subtopics:
  • Bubble Sort, Quick Sort, Heap Sort
  • Linear Search, Binary Search
  • String Matching (KMP, Rabin-Karp)
Graph Algorithms

We studied graph algorithms like DFS, BFS, Dijkstra’s, Bellman-Ford, Floyd-Warshall, and Minimum Spanning Tree algorithms (Prim’s, Kruskal’s).

Subtopics:
  • DFS and BFS
  • Dijkstra’s Shortest Path
  • Bellman-Ford, Floyd-Warshall
  • Prim’s and Kruskal’s Algorithms
Real-Time Applications

This course bridges the gap between theoretical understanding and practical application in fields like Smart City Design, sorting and searching, tree data structures, and competitive programming.

Subtopics:
  • Smart City Design: Optimizing road networks
  • Sorting and Searching for quick data processing
  • Tree Data Structures in file systems and compilers
  • Competitive Programming and Problem Solving
